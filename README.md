## Description

This plugin is essential for marketers, sales professionals, and anyone who wants to track users, manage deals, and analyze customer data through their store based on user interests.

WP User Insights Exporter provides a single page with a professional table that includes:

- User ID
- Name
- Email Address
- Country
- Phone Number
- Number of WooCommerce Orders
- Total Amount Spent

With just a few clicks, you can easily export all this data into a clean, organized table in Google Sheets or use it directly in Zoho Bigin or Zoho CRM.

The data exported from this plugin helps you:

- Manage your sales pipeline easily.
- Follow up with interested customers.
- Track potential leads.
- Close new deals.
- Organize your users inside Zoho Bigin or Zoho CRM efficiently.

By spending just one or two focused hours organizing and analyzing your customer data, you can boost your future revenue and build a much more organized business workflow.

## Key Features

- **Full User Data Export:** Export all user records, not just the current pagination page.
- **Professional Table Display:** Clean and beautiful table layout with shadow and rounded corners.
- **Comprehensive User Insights:** Displays user ID, name, email, country, phone number, WooCommerce orders count, and total amount spent.
- **Seamless CSV Export:** Export all user data into a CSV file for use with Google Sheets, Zoho CRM, Zoho Bigin, and more.
- **Sales Pipeline Optimization:** Easily organize and manage customers and leads inside your CRM based on actual user data.
- **Boost Deals and Revenue:** Quickly identify your top customers and leads to close more deals and grow your business.

## Installation

1. Upload the plugin files to the `/wp-content/plugins/wp-user-insights-exporter` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the plugin page to view and export user insights.

## Frequently Asked Questions

### Does this plugin work with both WordPress and WooCommerce?
Yes, WP User Insights Exporter works with both WordPress and WooCommerce, providing detailed insights based on user activities and order data.

### Can I export all users, not just the ones on the current page?
Yes, the plugin exports **all users** in the table, not just the users displayed on the current pagination page.

### Can I use the exported data in tools other than Google Sheets and Zoho?
Yes, the data is exported in a standard CSV format that can be imported into any compatible CRM or spreadsheet tool.

### How can this plugin help me close more deals?
By analyzing and organizing your customer data efficiently, you can better understand your audience, follow up with interested customers, and move leads through your sales pipeline more effectively.
